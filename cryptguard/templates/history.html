About Methods

What methods of encryption does CryptGuard identify?

Atbash Cipher:

The Atbash cipher is a particular type of monoalphabetic cipher originally used to encrypt the Hebrew alphabet. It can be adapted for all alphabets by reversing the order of the letters; the first letter becomes the last letter, the second letter becomes the second to last, and so forth.

XOR Cipher:

In the XOR cipher, the plaintext is combined with a key using the XOR operation (exclusive or). It is particularly notable for its use in modern computer operations, and when the key is the same length as the plaintext, this method is known as the Vernam cipher or one-time pad, which is theoretically unbreakable.

Rail Fence Cipher:

The Rail Fence Cipher is a form of transposition cipher that writes the message in a zigzag pattern on an imaginary fence, thus changing the order of the characters. The message is then read off in rows.

ROT13:

ROT13 is a specific implementation of the Caesar cipher, which uses a shift of 13. It means that it replaces a letter with the 13th letter after it in the alphabet. ROT13 is its own inverse, which simplifies encoding and decoding by the same operation.

Scytale:

SHA-256 and SHA-3(512) are popular members of the SHA-2 family. These are widely used for data integrity verification and digital signatures.

Reverse Cipher:

The Reverse cipher simply reverses the plaintext to create the ciphertext. It is extremely weak on its own but can be combined with other ciphers to add an extra step to the encryption process.

Beaufort Cipher:

The Beaufort Cipher is similar to the Vigenère cipher but uses a different method of creating the keystream. It effectively uses the same alphabet table as Vigenère but with a reversed decryption and encryption process.

Vigenère cipher:

The Vigenère cipher is a method of encrypting alphabetic text where each letter of the plaintext is encoded with a different Caesar cipher, whose increment is determined by the corresponding letter of another text, the key.

The Caesar cipher:

The Caesar cipher is a type of symmetric key encryption, meaning the same key (the shift value) is used for both encryption and decryption. The key, in this case, is the number of positions each letter is shifted.

Base64:

Base64 is a binary-to-text encoding scheme. It represents binary data in a printable ASCII string format by translating it into a radix-64 representation.
